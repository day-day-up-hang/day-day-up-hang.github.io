  • 右键菜单图标->运行->输入cmd->在命令提示符窗里输入“DISKPART”
  • 在弹出的DISKPART窗口中输入“LIST DISK”
  • 然后依次输入以下命令：（以磁盘2为SD卡为例）

SELECT DISK 2：选中磁盘2。

CLEAN：删除整个磁盘数据。

CREATE PARTITION PRIMARY：创建主磁盘分区。

ACTIVE：激活磁盘分区。

FORMAT FS=FAT32 QUICK：以FAT32格式快速格式化磁盘分区。
